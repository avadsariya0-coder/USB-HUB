# USB HUB
Custom 4-port USB-C HUB
___________________________________________________________________________________________________________________________________________________________________________

## Overview

 USB HUB is a custom designed USB HUB and is built using the SL2.1A USB HUB controller and USB-C Connectors based on USB-2. The project is made using KICAD for schematic and PCB and Fusion 360 for the 3D designing.

Unlike the another readymade hubs this USB HUB is made from the scratch from designing schematic to the PCB routing to the 3D designing.

This HUB allows 1 USB-C connection to expand into 4 USB-C port. It is useful for connecting multiple devices to 1 device.

___________________________________________________________________________________________________________________________________________________________________________

## Inspiration

The inspiration for designing USB HUB with USB-C connectors comes from the lack of the ports in the laptop. Modern laptop comes with very few ports which makes difficult to connect to the multiple devices at once.While most of the readymade USB hub comes with the USB-A connectors which is not available in Modern Macbooks and another laptops. To solve that .To fix the issue I came up with the USB HUB containing 1X USB-C upstream port and 4X USB-C downstream port.

___________________________________________________________________________________________________________________________________________________________________________

## Features

• 1 USB-C upstream port

• 4 USB-C downstream ports

• SL2.1A USBHUB controller

• USB-C powered

• Lightweight and portable

• Transparent enclosure
